O que é linux em si?

Para começar, “Linux” é o nome dado a uma das partes mais importantes de um sistema operacional: O Kernel. O kernel é responsável por realizar a conexão entre a parte lógica e a parte física. Realizar a conexão entre as peças do computador(disco rígido, memória RAM, processador, barramentos, entradas, etc.) e os programas, drivers, códigos, algorítimos… tudo! O kernel Linux foi desenvolvido pelo programador finlandês muito inteligente e revolucionário chamado Linus Towarlds e teve sua primeira versão lançada em 17 de setembro de 1991. O kernel Linux foi desenvolvido nas linguagem C e Assembly.

Então… Sim! O “Linux” não é um sistema operacional como o Windows ou o MAC OS, que por padrão possui uma interface gráfica com varias opções e coisas para interagir, mas sim um kernel! Um negócio que se permite interagir apenas com linha de comando em uma interface preta e nada convidativa! O Linux é aquilo que vai ficar posicionado na base, e o sistema operacional será construído em cima dele. E mais: sistemas operacionais Linux são chamados de Distribuições( ou Distros, se você preferir), e por quê? Porque o kernel Linux é gratuito então o que deriva dele também deve ser gratuito e consequentemente DISTRIBUIÇÕES do mesmo, com exceções de algumas empresas….

Como são as distribuição linux?

Como dito anteriormente, as distribuições são os sistemas operacionais construídas em cima do kernel Linux. E existem muuuuuuuuuitas distribuições Linux no mundo, com suas próprias características, funcionalidades, aparências, etc. De uma forma que você possa entender vovó: a aparência do sistema operacional Linux não é o que define qual á distribuição. Você pode muito ter a chance de ver uma distribuição que possui aparência semelhante ao Ubuntu mas… que não é o Ubuntu. O Linux na sua versão mais crua é apenas uma tela preta no qual você digita comandos grandes e complexos. Sem beleza, ponteiro do mouse, papel de parede… apenas uma tela preta com um tracinho piscando.Antes tudo era escuridão.

O que vai definir a aparência são os chamados Desktop Enviroments que traduzido para o nosso idioma significa Ambiente Desktop. São esses conjuntos de programas, binários, bibliotecas e outras complexidades que definem a aparência do sistema operacional e melhoram drasticamente a experiência do usuário. E se você souber como, você pode deixar a distribuição linux que seja com a aparência do que você quiser, instalando o Ambiente Desktop do seu interesse. Aqui alguns:

  3. XFCE

    4. O ambiente XFCE é um dos ambiente mais leves e mais customizáveis disponíveis para as distribuições linux. Se você busca estabilidade, fluidez, leveza e personalizações, então o XFCE foi feito para você.

OK, mas afinal, por que usar um sistema Linux?

Caro leitor, essa sua pergunta possui inúmeras respostas. Dentre algumas, tenho toda certeza que você já ouviu “É mais seguro que o Windows”, “Não pega vírus”, “É mais leve”, mas sabe… O que me chamou, me atraiu para o mundo Linux, não foram essas frases que mencionei, mas sim o fato de que os sistemas Linux favorecem a minha privacidade e me fazem me sentir livre do controle ou do acesso de alguma empresa multi-milionária. Outra coisa foi que utilizar Linux me deixa mais “diferente” das outras pessoas, visto que ainda não possui a devida popularidade do sistema da Microsoft e é bem raro encontrar alguém que utiliza um sistema livre.

Os sistemas Linux gerenciam melhor o uso dos componentes do meu computador. O Kernel possui uma maneira de gerenciar a memória RAM que é muito interessante e funcional para mim, que é transformando uma parte da memória usada em memória cache para deixar mais rápido e fluído a abertura de aplicativos; Então assim: mesmo que seu sistema esteja consumindo 80% da sua memória RAM, ele ainda estará suave e ainda vai poder abrir aplicativos sem muitos problemas , se você possuir a memória de troca, melhor ainda! Falando um pouco da memória de troca, o chamado “Swap” é uma memória virtual que será utilizada como uma RAM de emergência quando o seu sistema estiver utilizando muita memória. O sistema Windows também possui algo assim, mas não tem a mesma eficiência do existente no Linux. Se você deseja estudar um pouco mais sobre o manuseio da memória no Kernel Linux, acesse esse site : https://www.linuxatemyram.com/

Outro fator que eu gosto muito no sistema Linux é a sua filosofia Open Source. O código é aberto, livre dos bloqueios de licensa uma empresa multi-milionária, gratuito e qualquer um que tiver conhecimento pode editar e distribuir para outras pessoas baixarem, contanto que seja de graça(com a exceção de algumas empresas grandes que existem por ai que alugam seus sistemas linux, mas isso é um tópico para outro artigo futuro). Se você for um expert na área da programação e engenharia da computação, você pode criar seu próprio sistema Linux e utilizar sua cara e seu nome como logo! HAHAHA. Atualmente a grande maioria dos aplicativos disponíveis no sistema Windows estão disponíveis nos repositórios das distribuições Linux, ou até mesmo no próprio site do aplicativo tem lá uma versão para Linux, só basta que você vá atrás.

Para concluir, utilizar Linux é uma escolha completamente sua, e eu recomendo que você utilize por uma questão de aprendizado e experiência. Tente passar 5 dias com uma distribuição Linux, tenha problemas, vá atrás de resolver os problemas, aprenda mais e esse conhecimento/experiência ficará na sua memória para sempre. Se você não gostar de utilizar Linux, não tem problema, volte para o Windows ou o MacOS, ninguém vai lhe engolir por causa da sua decisão. Obrigado pela atenção. Artigo por José Isac
